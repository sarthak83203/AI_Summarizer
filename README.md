# 🧠 AI Summarizer Chrome Extension

A Chrome Extension that extracts article content from webpages and generates AI-powered summaries using the Groq API.

---

## 🚀 Installation Guide

### 1. Download the Project

Download the project as a **ZIP file** from this repository.

### 2. Extract the ZIP File

Extract the downloaded ZIP file to any folder on your computer.

### 3. Open Chrome Extensions

Open Google Chrome and go to:

chrome://extensions

### 4. Enable Developer Mode

Turn **ON Developer Mode** from the top-right corner.

### 5. Load the Extension

Click **Load unpacked** and select the extracted project folder.

The extension will now be installed in your browser.

---

## 🔑 Get Your Groq API Key

1. Go to the Groq console:

https://console.groq.com/keys

2. Create a new API key.
3. Copy the generated API key.

---

## ⚙️ Add the API Key to the Extension

After installing the extension:

1. The **Options page** will open automatically.
2. Paste your **Groq API Key** in the input field.
3. Click **Save Settings**.

Your extension is now ready to use.

---

## 📖 How to Use

1. Open any **article webpage**.
2. Click the **AI Summarizer extension icon**.
3. Select a **summary type**:

   * Brief
   * Detailed
   * Bullet Points
4. Click **Summarize**.

The extension will extract the article content and generate an AI summary.

* Extracts article text from webpages

* Multiple summary formats
* Copy summary to clipboard
